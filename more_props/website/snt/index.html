Slither IO

Slither IO est un jeu où le joueur incarne un ver de terre qui doit grossir en mangeant des nutriments (orbes colorées) ou les autres joueurs.
Le ver meurt s'il touche le corps d'un ver ennemi ou atteint la limite du monde.
Pour manger un ver ennemi, la tête de celui-çi doit toucher le corps du joueur, le ver ennemi se transformera alors en orbes que le joueur peut manger.

Paper Minecraft

Paper Minecraft est un clone en 2D du jeu minecraft. Il a été fait sur Sratch par Griffpatch.
Le jeu est assez similaire à minecraft dans la mesure où c'est un jeu en 2D avec des fonctionnalités limitées
Il a pour particularité d'avoir un système de sauvegarde, en effet, vous pouvez générer un code à copier et vous n'avez qu'à cliquer sur "Load Game" et coller le code pour continuer votre partie.

Krunker IO

Krunker IO est un FPS jouable sur navigateur. Il possède différentes maps et skins d'armes.
Vous pouvez choisir une classe qui définira quel arme vous jouez.
Exemple : Si vous choisissez "Triggerman" vous aurez un fusil d'assault.
