A propos d'Epic WebGames

Epic Web Games est un site permettant de jouer directement aux meilleurs jeux en ligne au même endroit.
Vous n'aurez même plus besoin de faire une recherche internet !
Epic Web Games est l'endroit où vous pouvez trouver les meilleurs jeux en ligne comme Slither IO, Paper Minecraft ou encore Krunker IO.
Ce site web a été développé par Maxime et contient des technologies futuristes permettant l'intégration de pages web directement dans ce site!
Vous trouverez çi-dessous les instructions permettant d'utiliser de façon optimale ce site.

Plein écran :
Pour une meilleure expérience de jeu vous pouvez utiliser le mode plein écran. Pour cela il vous suffit d'utiliser F11
ou de cliquer sur le bouton se trouvant en dessous du jeu.

Compte Scratch :
Afin d'accéder aux fonctionnalités multijoueur de certains jeux Scratch, vous avez besoin de connecter votre compte Scratch.

Erreur de chargement :
Si pour une quelconque raison un jeu ne veut pas charger, vous pouvez y accéder via un bouton en bas de la page, au dessus de du jeu


Blocages :
Certains de ces jeux peuvent être bloqués par le proxy du réseau sur lequel vous vous trouvez (par exemple dans un établissement scolaire)
Si c'est le cas, alors il n'y a pas grand chose que vous puissiez faire à moins d'avoir des talents en hacking...
