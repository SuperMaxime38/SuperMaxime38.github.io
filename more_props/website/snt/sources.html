Epic Games Epic Games est un lieu où on peut acheter des jeux, c'est ce site qui m'a inspiré le design de celui là

Slither IO Un des jeux disponible sur mon site (Voir slither io)

Scratch Ce site permet de créer des "programmes" sous forme de blocs, un des jeux disponible est fait sous scratch (Voir paper minecraft)

Krunker IO Un des jeux disponible sur mon site (Voir krunker io)

MDN MDN (Mozilla Developper Network) est un site où on peut trouver des références de code HTML et CSS avec des exemples


Les outils utilisés :

VSCode Visual Studio Code est un IDE (Environnement de Développement Intégré) qui permet de coder plus facilement en HTML, CSS, JAVASCRIPT, etc...

Remove BG remove.bg est un site permettant de retirer les fond des images (comme les fond blancs par exemple) pour ne laisser qu'un fonc transparent

Photofiltre Photofiltre est un outil de retouche photo qui m'a permis de retirer les pixels de transition entre le fond blanc d'une image et l'image (à cause de la compréssion).
